컴퓨터와 다음과 같이 주사위게임을 하는 프로그램을 작성하시오.

- 1 – 6의 눈이 있는 주사위 5개를 사용 함.
- 사용자는 처음 포함 주사위를 3번 던질 수 있음
- 사용자는 매번 던질 때 마다, 다시 던질 주사위 개수 및 위치를 선택할 수 있음. 즉, 2번째와 3번째는 어떤 주사위들을 다시 던질지 선택할 수 있고, 이 경우 다시 던지는 주사위는 값이 바뀔 수 있고, 다시 던지지 않는 주사위들은 값이 바뀌지 않음
- 컴퓨터 역시 3번 주사위를 던질 수 있음
- 컴퓨터의 행동은 다음과 같음
- 5개의 주사위의 숫자가 모두 같다면 더 이상 던지지 않는다.
- 5개의 주사위의 숫자가 모두 다르다면, 가장 높은 숫자 2개만 고정시키고 나머지 3개를 다시 던진다. 단, 연속된 숫자가 나온 경우 더 던지지 않는다.
  (예> 주사위가 1,2,3,4,6가 나왔다면 4,6이 나온 주사위는 고정시키고 1,2,3이 나온 주사위 3개만 다시 던짐. 1,2,3,4,5나 2,3,4,5,6이 나오면 더 던지지 않음)

- 동일한 숫자가 나온 주사위들이 있다면, 해당 주사위들은 고정시키고 나머지 주사위들만 다시 던진다.
(예> 1,3,3,5,6이 나왔다면 3이 나온 주사위 2개는 고정 시키고 나머지 1,5,6이 나온 주사위 3개만 다시 던짐)

(예> 2,4,5,5,5가 나왔다면 5가 나온 주사위 3개는 고정 시키고 나머지 2,4가 나온 주사위 2개만 다시 던짐)

(예> 2,5,5,5,5가 나왔다면 5가 나온 주사위 3개는 고정 시키고 나머지 2가 나온 주사위 1개만 다시 던짐)

- 동일한 숫자가 나온 주사위가 여러 개면 동일한 숫자의 개수가 많은 주사위를 고정시키고 나머지를 다시 던짐
(예> 2,2,2,5,5가 나왔다면 2가 나온 주사위 3개는 고정시키고 나머지 5가 나온 주사위 2개만 다시 던짐)

- 동일한 숫자가 나온 주사위가 여러 개이고, 동일한 숫자의 개수가 같다면 숫자가 큰쪽을 고정시키고 나머지를 다시 던짐
(예> 2,2,5,6,6이 나왔다면 6이나온 주사위 2개는 고정시키고 나머지 주사위 3개만 다시 던짐

승패를 정하는 패의 순위는 다음과 같음
5개가 모두 같은 숫자 (예> 5,5,5,5,5)
4개가 모두 같은 숫자 (예> 4,4,4,4,3)
3개가 같은 숫자이고 나머지 2개가 같은 숫자 (예> 3,3,3,2,2)
5개가 연속적인 숫자 (예> 1,2,3,4,5 또는 2,3,4,5,6)

2개가 같은 숫자이고 나머지 3개가 다른 숫자 (예> 1,1,2,3,4)
5개 모두 다른 숫자 (예> 1,2,4,5,6)
순위가 같은 패를 가진 경우, 동일한 패의 숫자가 큰 쪽이 이김
1,1,1,1,1과 5,5,5,5,5의 경우 5,5,5,5,5 승리
순위가 같은 패를 가지고, 동일한 패의 숫자가 같다면 나머지 숫자 중 제일 큰 수가 큰 쪽이 이김
1,1,1,1,2와 1,1,1,1,3의 경우 1,1,1,1,3이 승리
1,1,1,4,5와 1,1,1,2,6의 경우 1,1,1,2,6이 승리
